## Quick Tips for AI Generation

### For DALL-E 3 (ChatGPT)
- Upload this file and ask: "Generate product image for GLASS-001 using the prompt provided"
- Download as 1024x1024px
- Save as `SKU.jpg`

### For Midjourney
- Add `--ar 1:1 --v 6` to the end of each prompt
- Use `--quality 2` for higher detail
- Example: `[prompt] --ar 1:1 --v 6 --quality 2`

### For Stable Diffusion
- Use model: SDXL or SD 1.5
- Resolution: 1024x1024
- Steps: 30-50
- CFG Scale: 7-9
- Add negative prompt: `blurry, low quality, distorted, watermark, text`

### Batch Generation Tips
1. Generate all 24 images in one session for consistency
2. Keep the same style/lighting across all products
3. Use the verification script after uploading: `./scripts/verify-product-images.sh`
